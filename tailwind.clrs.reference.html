Clrs.cc Palette for TailwindCSS

Prototype websites faster with a rich colour selection from designer Adam Morse, of London

The Clrs.cc skin classes are designed "for fast prototyping" on the web.

Their website includes A11y friendly combinations, npm installation instructions, and the original Colours.css package this Tailwind colour configuration was based on.
